## Getting date
DATE=$(date +%Y%m%d)

## Create bucket with current date if just name exists.
aws s3api create-bucket --bucket "gpavelar" --region ap-southeast-2 --create-bucket-configuration LocationConstraint=ap-southeast-2 || aws s3api create-bucket --bucket "gpavelar${DATE}" --region ap-southeast-2 --create-bucket-configuration LocationConstraint=ap-southeast-2

## Output
http://gpavelar.s3.amazonaws.com/

## Upload file to s3 bucket
aws s3 cp ../../../../05class/aws_s3_rds/assets/index.html s3://gpavelar 
upload: ../../../aws_s3_rds/assets/index.html to s3://gpavelar/index.html

## Configure bucket name as static website
aws s3 website s3://gpavelar --index-document index.html

## Copy  `s3_bucket_policy.json` in the `assets` folder for my folder
## Edit resource name to my name.

## Put bucket policy
aws s3api put-bucket-policy --bucket gpavelar --policy file://s3_bucket_policy.json

curl http://gpavelar.s3-ap-southeast-2.amazonaws.com/index.html
